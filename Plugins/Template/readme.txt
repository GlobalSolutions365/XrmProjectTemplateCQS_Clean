Project template is used to create new Plugin projects inside the \Plugins folder.

Put Xrm.Plugin.Template.zip into %ProgramFiles%\Microsoft Visual Studio\2022\<edition>\Common7\IDE\ProjectTemplates\<Language>\<Locale ID>
(for example C:\Program Files\Microsoft Visual Studio\2022\Enterprise\Common7\IDE\ProjectTemplates\CSharp\1033)

Afterwards it should be enable in "File\New Project".
